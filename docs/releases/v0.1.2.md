# OpenSupportAI v0.1.2

## English

This release improves day-to-day admin conversation operations without requiring a database migration.

Highlights:

- Enriched admin conversation list API with `status`, `assignee_type`, `q`, `limit`, and `offset` filters.
- Added `summary` and `pagination` objects to admin conversation list responses.
- Added contact labels, message counts, recent-message previews, and latest handoff status to each admin conversation item.
- Updated the admin console with status filters, search, refresh, queue metrics, recent-message previews, and failed-handoff visibility.
- Added API regression coverage for conversation summary/filter behavior.
- Updated README and API documentation.

Validation:

- `pnpm format:check`
- `pnpm lint`
- `pnpm typecheck`
- `pnpm test`
- `pnpm build`
- `pnpm exec prisma validate`
- `pnpm db:generate`

## 中文

这个版本改进管理台日常会话运营能力，不需要数据库迁移。

主要更新：

- 管理端会话列表 API 新增 `status`、`assignee_type`、`q`、`limit`、`offset` 筛选。
- 管理端会话列表响应新增 `summary` 和 `pagination`。
- 每个会话项新增联系人标签、消息数、最近消息预览和最新 handoff 状态。
- Admin Console 新增状态筛选、搜索、刷新、队列指标、最近消息预览和失败 handoff 可见性。
- 新增 API 回归测试，覆盖会话摘要和筛选行为。
- 更新 README 和 API 文档。

验证项：

- `pnpm format:check`
- `pnpm lint`
- `pnpm typecheck`
- `pnpm test`
- `pnpm build`
- `pnpm exec prisma validate`
- `pnpm db:generate`
