# OpenSupportAI v0.4.0

## English

This release adds deterministic agent-assist primitives for human support workflows.

Highlights:

- Added Prisma `ConversationInsight` storage and migration.
- Added admin conversation assist APIs:
  - `GET /v1/admin/projects/{project_id}/conversations/{conversation_id}/assist`
  - `POST /v1/admin/projects/{project_id}/conversations/{conversation_id}/assist`
- Generated insights include:
  - conversation summary
  - suggested replies
  - deterministic tags
  - metadata for message/tool/handoff context
- Added insight data to admin conversation detail responses.
- Added handoff analytics API:
  - `GET /v1/admin/projects/{project_id}/analytics/handoffs`
- Added tests for insight generation, tags, suggested replies, and handoff analytics.

Validation:

- `pnpm format:check`
- `pnpm lint`
- `pnpm typecheck`
- `pnpm test`
- `pnpm build`
- `pnpm exec prisma validate`
- `pnpm db:generate`
- `pnpm smoke:memory -- --help`
- `pnpm smoke:chatwoot -- --help`

## 中文

这个版本加入确定性的坐席辅助基础能力，服务于人工客服工作流。

主要更新：

- 新增 Prisma `ConversationInsight` 存储和 migration。
- 新增管理端会话 assist API：
  - `GET /v1/admin/projects/{project_id}/conversations/{conversation_id}/assist`
  - `POST /v1/admin/projects/{project_id}/conversations/{conversation_id}/assist`
- 生成的 insight 包含：
  - 会话摘要
  - 建议回复
  - 确定性标签
  - 消息、工具调用、handoff 上下文 metadata
- 管理端会话详情响应新增 insight 数据。
- 新增 handoff analytics API：
  - `GET /v1/admin/projects/{project_id}/analytics/handoffs`
- 新增 insight 生成、标签、建议回复和 handoff analytics 测试。
