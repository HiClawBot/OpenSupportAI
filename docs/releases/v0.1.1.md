# OpenSupportAI v0.1.1

## English

This patch release stabilizes Chatwoot handoff operations and adds release validation infrastructure.

Highlights:

- Chatwoot connection test from the admin console.
- Handoff diagnostics and retry actions in conversation detail.
- Chatwoot `conversation_status_changed` webhook sync.
- GitHub Actions CI for format, Prisma validation/generation, lint, typecheck, test, and build.
- Live Chatwoot smoke-test script: `pnpm smoke:chatwoot`.
- Updated README, API spec, Chatwoot adapter docs, Docker Compose docs, and release checklist.

Validation:

- `pnpm format:check`
- `pnpm lint`
- `pnpm typecheck`
- `pnpm test`
- `pnpm build`
- `pnpm exec prisma validate`
- `pnpm db:generate`

## 中文

这个补丁版本重点稳定 Chatwoot 人工转接运维能力，并补齐发布校验基础设施。

主要更新：

- 管理台支持 Chatwoot 连接测试。
- 会话详情显示 handoff 诊断信息，并支持失败 retry。
- 支持 Chatwoot `conversation_status_changed` webhook 状态同步。
- 新增 GitHub Actions CI，覆盖 format、Prisma validate/generate、lint、typecheck、test、build。
- 新增真实 Chatwoot 联调脚本：`pnpm smoke:chatwoot`。
- 更新 README、API 规格、Chatwoot adapter 文档、Docker Compose 文档和发布检查清单。

验证项：

- `pnpm format:check`
- `pnpm lint`
- `pnpm typecheck`
- `pnpm test`
- `pnpm build`
- `pnpm exec prisma validate`
- `pnpm db:generate`
