# OpenSupportAI v0.1.3

## English

This patch release hardens local/self-host operations before the larger productionization work.

Highlights:

- Added configurable in-process fixed-window API rate limiting.
- Added standard `rate_limited` responses and `x-ratelimit-*` headers.
- Added `pnpm smoke:memory` for memory-mode end-to-end smoke validation.
- Improved admin console API errors by displaying backend code, message, and request id.
- Added favicon links for the admin console and demo app.
- Updated README, API docs, and release checklist.

Validation:

- `pnpm format:check`
- `pnpm lint`
- `pnpm typecheck`
- `pnpm test`
- `pnpm build`
- `pnpm exec prisma validate`
- `pnpm db:generate`
- `pnpm smoke:memory -- --help`
- `pnpm smoke:chatwoot -- --help`

## 中文

这个补丁版本在进入更大的生产化迭代前，先加强本地和自托管运维基础。

主要更新：

- 新增可配置的进程内固定窗口 API 限流。
- 新增标准 `rate_limited` 响应和 `x-ratelimit-*` 响应头。
- 新增 `pnpm smoke:memory`，用于内存模式端到端 smoke test。
- 管理台 API 错误提示会显示后端 code、message 和 request id。
- Admin Console 和 Demo App 新增 favicon。
- 更新 README、API 文档和发布检查清单。

验证项：

- `pnpm format:check`
- `pnpm lint`
- `pnpm typecheck`
- `pnpm test`
- `pnpm build`
- `pnpm exec prisma validate`
- `pnpm db:generate`
- `pnpm smoke:memory -- --help`
- `pnpm smoke:chatwoot -- --help`
